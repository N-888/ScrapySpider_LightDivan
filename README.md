#  Scrapy Паук для парсинга источников освещения

##  Описание проекта
Автоматический сбор информации об источниках освещения с сайта divan.ru

##  Возможности
- Сбор названий, цен и ссылок на товары
- Сохранение в JSON и CSV форматах
- Обработка 48+ товаров за 2 секунды

##  Технологии
- Python 3.13
- Scrapy Framework
- CSS-селекторы
